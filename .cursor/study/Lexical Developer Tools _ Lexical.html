Lexical Developer Tools

Extension

The easiest way to debug websites built with Lexical is to install the Lexical Developer Tools browser extension. It is available for several popular browsers: Now, if you visit a website built with Lexical (such as https://playground.lexical.dev/ ), you will see the Lexical panel in your browser Inspector window.
